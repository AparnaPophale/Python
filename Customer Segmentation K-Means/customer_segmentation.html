Project: Customer Segmentation with K-Means

This project performs customer segmentation using the K-Means clustering algorithm. It groups customers based on Age, Annual Income, and Spending Score a common task in marketing analytics for audience targeting and persona profiling.

Steps:

  1. Load and explore dataset
  2. Scale features using StandardScaler
  3. Use the Elbow Method to determine optimal K
  4. Train K-Means clustering model
  5. Visualize customer segments
  6. Export clustered dataset

Files:

  • customer_segmentation.ipynb main notebook
  • customers.csv input dataset
  • customers_segmented_output.csv model output
  • requirements.txt

# Select numerical features
features = ["Age", "Annual_Income", "Spending_Score"]
X = df[features]
In[11]:
# Scale features
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
In[12]:
# Determine optimal clusters using the elbow method
inertia = []
K = range(2, 8)

for k in K:
    km = KMeans(n_clusters=k, random_state=42)
    km.fit(X_scaled)
    inertia.append(km.inertia_)
In[13]:
plt.plot(K, inertia, marker="o")
plt.xlabel("Number of clusters")
plt.ylabel("Inertia")
plt.title("Elbow Method for Optimal K")
plt.show()
In[14]:
# Fit K-Means model
kmeans = KMeans(n_clusters=4, random_state=42)
df["Cluster"] = kmeans.fit_predict(X_scaled)
